## [1.0.0] - 2025-11-16

### Added
- Initial release of Celery plugin
- 15 slash commands for comprehensive Celery workflow
- 12 specialist agents for setup, development, and deployment
- 10 skills with templates, examples, and scripts
- Django, Flask, and FastAPI integration support
- Redis, RabbitMQ, and Amazon SQS broker support
- Flower monitoring integration
- Production deployment configurations
- Comprehensive documentation
