# Changelog

All notable changes to the FastMCP plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-10-25

### Added
- Initial FastMCP plugin release
- `/fastmcp:new-app` command for project initialization
- `fastmcp-verifier-py` agent for Python application validation
- Comprehensive FastMCP SDK documentation integration
- Support for tools, resources, and prompts development
- OAuth 2.1 authentication provider support
- Deployment configuration helpers
