# Changelog

All notable changes to the nextjs-frontend plugin will be documented in this file.

## [1.0.0] - 2025-10-31

### Added
- Initial release of nextjs-frontend plugin
- 6 commands: init, add-page, add-component, integrate-supabase, integrate-ai-sdk, search-components
- 6 agents: nextjs-setup, component-builder, page-generator, ai-sdk-integration, supabase-integration, ui-search
- 2 skills: deployment-config, tailwind-shadcn-setup
- Complete Next.js 15 App Router support
- AI SDK integration with streaming support
- Supabase integration for auth and database
- shadcn/ui component library integration
- TypeScript and Tailwind CSS setup
- Comprehensive documentation and examples
