# Design Inspiration Skill

Curated collection of top-tier website designs with pattern analysis and implementation guidance.

## Features

- **Curated Site Lists** - Best SaaS, marketing, e-commerce sites
- **Pattern Library** - Hero, navigation, pricing, social proof patterns
- **Color Palettes** - Dark mode, light mode, warm themes
- **Typography Scales** - Modular scales (1.25, 1.333)
- **Spacing Systems** - 8pt grid, section spacing

## Quick Start

```bash
# Load the skill
!{skill design-inspiration}

# Analyze a specific site
/analyze-design https://linear.app
```
