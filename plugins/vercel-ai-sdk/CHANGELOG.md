# Changelog

All notable changes to the Vercel AI SDK plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-10-24

### Added
- Initial release of Vercel AI SDK plugin
- Support for TypeScript, JavaScript, and Python
- Project setup command with interactive configuration
- Streaming implementation support
- Tool calling features
- Multi-provider support (OpenAI, Anthropic, etc.)
- Verifier agents for TypeScript, JavaScript, and Python
- Comprehensive documentation and examples
