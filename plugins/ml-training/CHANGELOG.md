# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-01

### Added
- Initial plugin scaffold
- Plugin directory structure
- Plugin manifest (plugin.json)
- 18 slash commands for ML training pipeline
- 14 specialized agents for cloud GPU workflows
- 7 skills for training patterns and integrations
- MCP server configurations (Supabase, Context7)
- Two-tier dependency management (local vs. cloud GPU)
- Support for Modal, Lambda Labs, and RunPod platforms
- Integration with HuggingFace ecosystem (Transformers, PEFT, Accelerate)
