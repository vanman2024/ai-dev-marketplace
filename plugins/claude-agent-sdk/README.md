# Claude Agent SDK Plugin

Complete Claude Agent SDK plugin for building AI agents with TypeScript and Python support.

## Overview

This plugin provides comprehensive tooling for building AI agents using the Claude Agent SDK. It includes commands for project initialization, agents for validation, and support for both TypeScript and Python workflows.

## Commands

### `/claude-agent-sdk:new-app`
Create and setup a new Claude Agent SDK application with TypeScript or Python.

**Usage:**
```
/claude-agent-sdk:new-app my-agent
```

**Features:**
- Interactive project setup
- Language selection (TypeScript/Python)
- Latest SDK version installation
- Example code generation
- Project structure creation
- Automatic validation

## Agents

### `claude-agent-verifier-ts`
Validates TypeScript Claude Agent SDK applications for correctness and best practices.

### `claude-agent-verifier-py`
Validates Python Claude Agent SDK applications for correctness and best practices.

## Installation

This plugin is part of the ai-dev-marketplace. It will be automatically available when the marketplace is loaded.
